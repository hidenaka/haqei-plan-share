# HaQei / アニメと易経 — 中間検証と今後の修正方針（全体統括レポート）

作成: 2026-07-20 ／ 統括: Claude（調査は codex 専門家4名 gpt-5.6-sol を並行実行）

## エグゼクティブサマリー（先に結論）

登録90人・+0.86人/日＝目標1000まで約1,062日（3年）。この3年ペースを1年に畳むには **2.9倍** の加速が要る。4名の専門家（ファネル／ショート／コンテンツ／ポジショニング）が独立に調査した結果、診断は驚くほど一点に収束した。

**律速は「登録ボタンへの導線（CTA配線）」ではない。ショートも本編も『一本で完全に閉じてしまい、"次を見る理由／登録する理由"が作品の中に残っていない』ことである。** ショート批評家とコンテンツ批評家が、まったく別の入口（数値台帳の分析 vs 実台本の精読）から同じ結論に到達したのが最も強い証拠。

さらに、現行の第一律速目標「0.036%→0.1%」は**数学的に不十分**（1年に畳むには約0.127%が必要）。そして判定の土台である計器（funnel.json / timeseries.jsonl）が**文字化けでパース不能**という盲点が見つかった。

→ 修正の柱は4つ: ①計器を直す（全判断の前提）②北極星の張り替え ③「閉じない」設計＝最大レバー ④冒頭の資格化＋ポジショニング微修正。

## 2. 4名の専門家診断（収束点）

### 収束① 律速の再定義（ファネル＋ショート＋コンテンツが一致）
現行台帳の rank1「ショート視聴→登録の変換（CTA配線で0.036%→0.1%）」は**方向は正しいが定義が狭い**。真の第一律速は **「ショート1,000再生あたりの登録獲得量＝作品として心を動かせているか」**。根拠：完視聴96-98%のショート2本でも登録ゼロ／いいね→登録率は本編15.1%・ショート11.9%でほぼ同じ（＝心が動いた後の登録率ではなく、心が動く人そのものが少ない）。

### 収束② 「閉じすぎ」が登録を殺している（ショート批評家＋コンテンツ批評家が独立一致・最重要）
- **ショート側**（sh38夬・sh40小過を精読）: 「誰も聞いていない報告会」で完全決着→視聴者は知的満足するが**次の一本を見る必要がない**。唯一の継続訴求は無言バッジのみ。
- **本編側**（ep01/03/17/37/38を精読）: 問いも世界観も完全に閉じ、次の一本で何が広がるか見せていない。ep38には唯一の横断線があるが「深追いはしませんが」と**自分から扉を閉じている**。
- → **修正: 幕6/S06 で主題には答えたうえで、"隣の卦・反対の人物・次の変化"を一つだけ開け、既存動画のエンドカードへつなぐ。**「逃げる」の次に「戻る」がある、と物語の言葉で示す。

### 収束③ 冒頭が遅い＝クリック時の約束との接続が遅れている
- **本編**: タイトルで来た作品・キャラの登場まで**31〜46秒**（最大離脱は中央値9秒）。最初の30秒は別の一般論を渡している。
- **ショート**: 作品名が音声で出るのは**8〜10秒後**→Shortsフィード上でしばらく「一般的な自己啓発動画」に見え、大型IPで呼んだファンに最初アニメを見せていない。
- → **修正: 最初の3〜5秒に「固有名（作品/キャラ）＋既知像の反転＋自分ごとの痛み」を同じ一文で接着する。**

### 収束④ 計器が壊れている（ファネル専門家が発見・P0）
- **funnel.json / timeseries.jsonl は不正Unicode（孤立サロゲート）でJSONパース不能**。時系列計器として機能していない。
- **7日窓（0.86人/日）と30日窓（1.37人/日）で結論が逆転**する。0.1%目標が「454日」か「362日」かが窓依存で決まる＝判定不能。
- growth-tick 内の律速説明が旧値（63,500/1,331）のまま。
- H35（CTA配線）コホートは処置群n=0で**判定不能**。「CTAは効かない」もまだ実証されていない。

### 収束⑤ ポジショニング＝易経は"入口"でなく"エンジン"（ポジショニング専門家）
- 実際に刺さっている需要は「易経」ではなく **「有名作品から現実の悩みを読み解く大人向けアニメエッセイ」**。易経を**クリック前の主役にすると入口が狭まる**。
- CTR約1%の主因は題材不足でなく **「アニメ考察タイトル × 和風アート/人生訓サムネ」のミスマッチ**（0.5秒で何の動画か伝わらない）。
- 【第N話・卦】suffix は人間には負荷（「36本見逃した」感・卦名は非易経層に読めない）。ただしH49/H50実験進行中なのでep64まで割付は変えず結果を待つ。
- 発見経路の現実性順: ①ショート直接登録（開いているが天井）②関連/ブラウズ（配信面は開くがクリック負け）③検索（ほぼ閉）④コレクション回遊（未開）。**ショート→本編送客を主ファネルにしない**（形式横断視聴は一般的でない）。

## 3. 統括の裁定（専門家間の緊張をどう畳むか）

- **切り抜き廃止→書き下ろし転換は「方向は正しいが、全面廃止と型の一本化は時期尚早」**（ショート批評家）。H51の公開実績は事実上1本で、しかも26時間0再生＋同題材近接投稿で汚染された標本。**"転換は成功した"と宣言できるデータはまだ無い。** null を勝利にも敗北にも読み替えない。
- **本編は"17倍の変換装置"だが流量2%。CTR施策単独では3%達成でも+0.37人/日で主戦場にならない。** 本編は「既存CTR実験の判定＋勝ち5本への集中」に絞り、新規投資を増やさない。
- **1日1本の書き下ろしショートは持続不能**（仕様自身が「2本は回らない」と認める）。→ ハイブリッド（本編と fact/素材を共有する独立再構成型。生の切り抜きは使わない）。

## 4. 数学リアリティチェック（1年で1000に畳む経路）

必要ペース = 残910 ÷ 365日 = **2.493人/日**（現0.86の2.9倍）。単独シナリオの必要倍率:

| シナリオ | 1年到達の必要水準 | 必要倍率 | 評価 |
|---|---|---|---|
| ①ショート変換のみ | **0.127%** | 3.54倍 | **最小倍率＝第一候補** |
| ②ショート流量のみ | 約19万再生/月 | 3.54倍 | バズ依存で不安定 |
| ③本編流量のみ | 約8,863再生/月 | 約8.6倍 | 計器欠・非現実 |
| ④外部チャネルのみ | +49登録/月 | 約224倍 | 単独主戦場にならない |

**最短経路＝掛け算**: H51推定（いいね率1.0% × いいね→登録11.9% = 登録率0.119%）＋流量+7〜27%。つまり **「変換改善（閉じない設計＋冒頭資格化）」を主軸に、勝ち型だけ流量を少し増幅** するのが数学的に最小の変更。

## 5. 今後90日の修正プラン（優先度順）

1. **【P0・全判断の前提】計器を直す** — funnel/timeseries のUnicode健全性ゲート（書込前にJSON parse必須化）＋7日/30日/lifetimeを分離した統一30日ファネル表（format×views×likes×shares×saves×subsGained）。効果は直接0人/日だが、1人/日級の誤配分を防ぐ。判定: 14日連続で全JSON parse成功。
2. **【最大レバー】"閉じない"設計を全ショート＋本編の幕6/S06に実装** — 主題は答える／卦の番号で位置を明示／隣の変化を一つ約束／既存動画エンドカードへ接続。定型CTAは不要。期待: いいね率1%再現なら+1.48人/日級。判定: d7いいね率≥1.0%かつ subs/view を併記。
3. **北極星を張り替える** — 「CTA配線で0.1%」→ **「short subs / 1,000 views ≥ 0.127%」**（書き下ろしショートのd7/d14コホートで測る）。0.1%では454日で1年に届かない。
4. **冒頭の資格化** — 本編幕1・ショートS01を「最初の3〜5秒に固有名＋反転＋自分ごと」へ改修。9秒離脱とフィード上の"一般自己啓発化"を同時に潰す。
5. **ハイブリッド制作で H51作品を10〜12本充填** — 週3本=書き下ろし旗艦／週2本=本編fact・卦対応・画像資産を再利用する独立再構成。1日1本ノルマで品質を削らない。判定してから宣言する。
6. **ポジショニング微修正（保守案）** — サムネを「作品固有シンボル＋4〜8字の葛藤語」へ／勝ったIPを3本単位のミニシリーズにクラスタ化／H49・H50終了後に話数・卦名をタイトル前面から概要欄・プレイリストへ退避。
7. **撤退線を置く** — TikTok/外部は「低コストのフック実験場」であって第二成長エンジンではない。専用フル制作はしない。本編は既存CTR実験を判定するだけで新規実験を開かない。判定超過3件・coord超過19件・M1停滞18日を先に片づける。

## 6. 最も自信のある主張 / 判断保留

**自信トップ3**
1. 律速はCTA単体でなく「ショートの反応品質＋登録獲得量」。いいね後の登録率が形式間でほぼ同じなのが最強の証拠。
2. 0.1%は十分条件でない。7日基準で必要率は約0.127%。
3. 最短経路はショート改善の掛け算（変換＋流量7〜27%増）。本編CTR・外部単独では桁が足りない。

**判断保留（データ不足）**
- 同一30日窓でのショート/本編/直接登録の完全内訳（計器修理で解消）。
- H51のいいね率1%が登録率0.119%へ再現するか（コホートが小さい）。
- 1日1本書き下ろしの長期持続性（制作コスト記録が無い）。
- TikTokが独立してフォロワーを増やす成長エンジンか（転換データ不足）。
