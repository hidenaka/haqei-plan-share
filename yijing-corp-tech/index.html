v2は「状態＝ファンダメンタルズ7軸／予測＝変化方向」だった。v3では市場が会社をどう織り込んでいるか（P/E・P/B・EV/EBITDA・価格モメンタム・織り込み期待）を状態に加え、状態の読みとバリュエーションの差＝ミスプライシング（expectations gap / variant perception）からフォワードの価格方向を予測・バックテストする。ダムダウンしない技術版。

状態 ファンダ7軸 ＋ 市場3軸 予測 ①状態変化 ＋ ②価格方向 橋渡し mispricing / variant perception 価格予測 private層＋研究に限定（金商法）

01状態モデルの拡張 ― 市場評価3軸を追加

既存のファンダメンタルズ7軸（①方向〜⑦市場）に、「市場が会社をいくらで、どんな期待で織り込んでいるか」の3軸を追加する。すべて snapshot時点で入手可能な指標（look-ahead厳守）。

軸指標（専門用語）読み
①〜⑦（既存）売上/セグメント推移・成長率・営業CF/投資CF・ROIC・FCF・負債資本比率・IRテキスト差分・経営者トーン・業界成長/シェアファンダメンタルズの状態
⑧ バリュエーションtrailing/forward P/E・P/B・EV/EBITDA・FCF利回り・配当利回り（自己時系列＆peer相対）割安(+)〜割高(-)＝margin of safety
⑨ 価格モメンタム直近6〜12M トータルリターン・対ベンチマーク相対強度・最大ドローダウン・ボラティリティ/β上昇(+)/下降(-)トレンド
⑩ 織り込み期待forward multipleが含意するimplied growth・アナリストコンセンサス・リビジョン方向期待過小(+)/priced for perfection(-)

「状態に株価を入れる」の意味：株価そのもの＋バリュエーション＋モメンタムは「市場という第三者が会社をどう評価しているか」という状態変数。ファンダの7軸（会社の実体）と、市場評価の3軸（他者の織り込み）を別レイヤーとして併記することで、次章の「実体と価格のズレ」を測れるようになる。

02二重ターゲット予測 ― 状態変化 ＋ 価格方向

予測対象を1つから2つへ。卦は両方の解釈レイヤーとして併記する。

ターゲット出力正解ラベル（バックテスト）
A：状態変化（既存）expand / contract / pivot / stall実際の事業の帰結（回復/転落/転換）
B：価格方向（新規）up / flat / down（＋確信度）snapshot→horizonのTSR符号・対ベンチマーク超過リターン

なぜ分けるか：「良い会社（状態expand）」と「上がる株（価格up）」は別物。既に割高なら状態が良くても価格は伸びない（織り込み済み）。逆に状態が悪くても十分に叩き売られていれば反発する。Bは A を入力の一部として使うが、A＝Bではない。

04バックテストの拡張 ― 価格方向の検定

状態の的中に加え、価格方向の的中とリターンの質を測る。

  • hit rate（価格方向）：up/flat/down の的中率を base rate（無情報の当てずっぽう）と比較。
  • 超過リターン：予測 up 群の対ベンチマーク（TOPIX/S&P500）アルファ、予測 down 群の劣後。
  • Information Coefficient（IC）：予測スコアと実現リターンの順位相関（Spearman）。
  • 過学習ゲート（既存）：design↔holdout の乖離。look-aheadゲート：バリュエーション/モメンタムの as_of ≤ snapshot。

統計的留保を明示：N=6 では検出力は皆無に近い。価格方向の的中率で優位を主張しない。ここでの価値は「配線が通り、IC/超過リターンを再現可能に計測する土台」。コーパス拡大（数十社）まで結論は保留（null を反証に使わない）。

05法的ガードレール（重要・金商法）― 価格予測はここに置く

価格方向の予測を持つと投資助言業（登録制）に近づく。レイヤーで隔離して安全側に倒す。

レイヤー置くもの出さないもの
public（公開・研究/教育）手法の的中率・IC・ケーススタディ・外れ分析（不特定多数・随時・オープン）個別銘柄の売買判断（買え/売れ/目標株価/時期）
private（自分用）個別銘柄の価格方向・ミスプライシング判定・自分の投資メモ公開・有料配信・会員限定の銘柄シグナル

線引き：「価格方向を予測し的中率を研究として公開」は教育・研究に収まる。規制の核は「特定銘柄の投資判断の助言 × 報酬 × 会員制」の同時成立。だから価格予測・売買判断はprivate層のみ、公開層は銘柄判断語ガード（買え/売れ/目標株価を機械検出して停止）を通す。有料化・会員制・個別銘柄配信に踏み込む前に弁護士確認（免責文だけでは該当性は消えない）。

結論

  1. 状態に市場評価3軸（バリュエーション・モメンタム・織り込み期待）を追加＝「株価を状態に入れる」を専門用語で実装。
  2. 予測を二重ターゲット（状態変化＋価格方向TSR）化し、橋渡しはミスプライシング行列（variant perception）。良い会社≠上がる株を分離。
  3. 価格予測・売買判断はprivate層＋的中率研究に限定し、公開層は銘柄判断語ガードで金商法の投資助言業を回避。次は6社の株価データでengineを拡張。
技術設計 v3 ／ 状態×市場評価→価格方向 ／ 本資料は研究・教育目的の設計であり、個別銘柄の投資助言ではない。価格予測機能は自己利用・研究用途に限定し、有料化/会員制/個別銘柄配信に踏み込む運用は弁護士確認が前提。
